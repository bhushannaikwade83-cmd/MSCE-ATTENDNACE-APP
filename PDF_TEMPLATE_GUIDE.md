# PDF Report Template Customization Guide

## Overview

You can now customize your attendance report PDF templates! The template system allows you to:
- Change colors, fonts, and layout
- Show/hide specific sections
- Add custom fields
- Customize headers and footers

## Quick Start

### Method 1: Direct Code Modification (Easiest)

Edit the file: `lib/services/pdf_template_config.dart`

Simply change the values in the `PdfTemplateConfig` class:

```dart
// Example: Change report title
static String reportTitle = 'My Custom Attendance Report';

// Example: Change header color to green
static PdfColor headerBackgroundColor = PdfColors.green700;

// Example: Hide student photos
static bool showStudentPhotos = false;
```

### Method 2: JSON Configuration (Coming Soon)

You can create a JSON file with your template settings and load it:

```json
{
  "reportTitle": "My Custom Report",
  "headerBackgroundColor": {"r": 0.0, "g": 0.5, "b": 0.0},
  "showStudentPhotos": false,
  "goodAttendanceThreshold": 80.0
}
```

## Available Customization Options

### Header Configuration

```dart
// Report title text
static String reportTitle = 'Attendance Report';

// Institute name (leave null to auto-detect)
static String? instituteName;

// Logo image path (optional)
static String? logoPath;

// Header colors
static PdfColor headerBackgroundColor = PdfColors.blue700;
static PdfColor headerTextColor = PdfColors.white;
static double headerFontSize = 24.0;
```

### Colors

```dart
// Present/Good attendance color
static PdfColor presentColor = PdfColors.green700;

// Absent/Poor attendance color
static PdfColor absentColor = PdfColors.red700;

// Warning/Medium attendance color
static PdfColor warningColor = PdfColors.orange700;

// Primary accent color
static PdfColor primaryColor = PdfColors.blue700;

// Table colors
static PdfColor tableHeaderBackgroundColor = PdfColors.grey200;
static PdfColor tableBorderColor = PdfColors.grey300;
static PdfColor? rowAlternateColor = PdfColors.grey100;
```

### Layout

```dart
// Page margins (all sides)
static double pageMargin = 40.0;

// Spacing between sections
static double sectionSpacing = 20.0;

// Table cell padding
static double tableCellPadding = 8.0;

// Border radius for containers
static double borderRadius = 8.0;
```

### Content Visibility

```dart
// Show/hide sections
static bool showStudentPhotos = true;
static bool showSubjectBreakdown = true;
static bool showDailyDetails = true;
static bool showStatisticsSummary = true;
static bool showEntryExitTimes = true;
static bool showTotalHours = true;
static bool showAttendancePercentage = true;
```

### Footer

```dart
// Show footer
static bool showFooter = true;

// Footer text
static String footerText = 'Generated by Attendance Management System';

// Footer colors
static PdfColor footerTextColor = PdfColors.grey600;
static double footerFontSize = 10.0;
```

### Thresholds

```dart
// Minimum attendance percentage for "good" status
static double goodAttendanceThreshold = 75.0;
```

## Color Reference

Common PDF colors you can use:

```dart
PdfColors.blue700    // Blue
PdfColors.green700   // Green
PdfColors.red700     // Red
PdfColors.orange700  // Orange
PdfColors.grey700    // Grey
PdfColors.black      // Black
PdfColors.white      // White
PdfColors.purple700  // Purple
```

Or create custom colors:

```dart
static PdfColor customColor = PdfColor.fromInt(0xFF1E88E5); // Custom blue
```

## Resetting to Default

If you want to reset all settings to default:

```dart
PdfTemplateConfig.resetToDefault();
```

## Export/Import Template

### Export Current Template

```dart
final templateJson = PdfTemplateConfig.toJson();
// Save this JSON to a file or Firestore
```

### Import Template

```dart
final templateJson = {
  'reportTitle': 'My Report',
  'headerBackgroundColor': {'r': 0.0, 'g': 0.5, 'b': 0.0},
  // ... other settings
};
PdfTemplateConfig.fromJson(templateJson);
```

## Tips

1. **Test Changes**: After modifying the template, generate a test report to see the changes
2. **Color Contrast**: Ensure text colors have good contrast with background colors
3. **Font Sizes**: Adjust font sizes based on your needs (default is 10pt for body text)
4. **Margins**: Increase margins if you need more white space
5. **Performance**: Hiding sections you don't need can make reports generate faster

## Need Help?

If you need more customization options, you can:
1. Modify `pdf_template_config.dart` to add new options
2. Edit `pdf_export_service.dart` to change the PDF layout structure
3. Contact support for advanced template features
