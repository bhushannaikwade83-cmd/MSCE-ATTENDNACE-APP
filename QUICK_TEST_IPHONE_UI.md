## 🎯 **3 Simple Steps:**

### **Step 1: Run Web App**
Open terminal in your project and run:
```bash
flutter run -d chrome
```

### **Step 2: Open Chrome DevTools**
- Press `F12` (or right-click → Inspect)
- OR press `Ctrl+Shift+I`

### **Step 3: Enable iPhone Mode**
- Click the **device icon** (📱) in DevTools
- OR press `Ctrl+Shift+M`
- Select **iPhone 12 Pro** (or any iPhone)

**That's it!** Your app now shows in iPhone dimensions! 🎉
